# Asistente sobre Cambio Climático

Proyecto final desarrollado en Python usando Flask e Inteligencia Artificial

Esta aplicación web informa sobre el cambio climático y permite al usuario hacer preguntas relacionadas con el tema.

El sistema responde de dos formas:
- *Respuestas predefinidas* para preguntas comunes.
- *Inteligencia Artificial* cuando la pregunta no está contemplada previamente.

Tecnologías utilizadas:
- Python
- Flask
- HTML, CSS y JavaScript(un poquito)
- API de Groq (LLM)

Funcionalidades:
- Información educativa sobre el cambio climático
- Asistente virtual interactivo
- Loader animado mientras la IA responde
- Validación y control de errores
- Respuestas limitadas exclusivamente al cambio climático
